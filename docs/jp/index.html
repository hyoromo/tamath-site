『一筆書きで数式を作る脳トレゲーム』

このゲームは一筆書きで足し算、引き算、掛け算の計算をするパズル的なゲームです。
操作は簡単！画面に表示される数字を一行でつないでいくだけ。
シンプルなルールで誰でも簡単に遊べますが、レベルが上がるにつれ難易度は高まっていきます。
脳を思い切り働かせて早く問題を解こう！

3つの難易度

【EASY】足し算と引き算の簡単な計算問題
【NORMAL】掛け算だけの計算問題
【HARD】足し算/引き算/掛け算に加え、式の桁数が増えた計算問題
