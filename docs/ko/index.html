일휘식 수식 퍼즐 게임

이 게임은 한 휘로 덧셈, 뺄셈, 곱셈 계산을 하는 퍼즐 게임입니다.
조작은 간단! 화면에 표시된 숫자를 한 줄로 이어나가기만 하면 됩니다.
규칙이 간단하여 누구나 쉽게 즐길 수 있지만, 레벨이 올라갈수록 난이도는 높아집니다.
뇌를 매진해 빨리 문제를 풀어보자!

3 가지 난이도

【EASY】 덧셈과 뺄셈의 간단한 계산 문제.
【NORMAL】 곱셈 계산문제만.
【HARD】 덧셈/뺄셈/곱셈에 숫자 자릿수가 늘어난 계산 문제.

랭킹
